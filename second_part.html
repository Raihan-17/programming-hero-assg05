Question-1: What are the different ways to select an element in the dom?

Using getElementById.
Using getElementsByClassName.
Using getElementsByTagName.
Using querySelector.
Using querySelectorAll.
So far , i found the getElementById is the mostly used one.

Question-2: What is the difference between innerHTML,innerText and textContent?

Answer : We use innerHTML when we need HTML content.It gets or sets the HTML content inside an element, including tags.
We use innerText when we need visible text.It gets or sets the visible text content, as well as CSS styles
We use textContent when we need all text, regardless of visibility and it ignores css styles.

Question-3: What is event delegation in the dom?

Answer : Event delegation in the DOM is a JavaScript technique where a single event listener is attached to a parent element instead of individual child elements. It works by taking advantage of event bubbling. If a child element is clicked , the event bubbles up to the parent and then the parent can handle it.

Question-4: What is event bubbling in the dom?

Answer :When an event , like a click starts on the element we interact with and then bubbles up to its parent elements, one by one, until it reaches the top of the page, this process is called event bubbling in the DOM. It lets us handle events on parent elements without adding event listeners to each one of them.
